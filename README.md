### Installation
1. Clone the repository:
   
   ```bash
   https://github.com/avivt10/order-management-server.git
   ```
2. Navigate to the project directory:
   
   ```bash
   cd order-management-server
   ```
3. Install dependencies:
   
    ```bash
   npm install
    ```
4. Finally, start the development server:

   ```bash
   nodemon
   ```
